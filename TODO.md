# AI 阅读助手 — 功能清单

## 已实现

### 核心功能
- [x] 页面内容提取 — Readability.js 提取正文，失败时回退到 `body.innerText`
- [x] 总结页面 — 一键生成页面摘要（3-5 个要点）
- [x] 翻译页面 — 将页面内容翻译为中文
- [x] 提取关键信息 — 列出重要事实、数据、观点
- [x] 自由问答 — 基于页面内容的对话式问答
- [x] 流式输出 — SSE 逐字显示 AI 回复
- [x] Markdown 渲染 — marked.js 渲染 AI 回复
- [x] 推理过程展示 — 支持 DeepSeek 等推理模型的思考过程，可折叠显示

### 设置与配置
- [x] API Key 配置 — options 页保存到 `chrome.storage.sync`
- [x] 自定义 API 地址 — 支持 OpenAI 兼容接口（默认 DeepSeek）
- [x] Key 输入遮盖 — `type="password"` 安全处理
- [x] 自定义 System Prompt — 追加到默认 prompt 之后，个性化 AI 回答风格
- [x] 模型列表获取 — 点击刷新按钮自动获取可用模型列表
- [x] 设置导入/导出 — 一键备份和恢复全部配置（JSON 格式）

### UI 交互
- [x] 侧边栏 UI — Chrome Side Panel 面板
- [x] 快捷操作按钮 — 总结 / 翻译 / 关键信息
- [x] 对话气泡 — 用户消息与 AI 回复区分显示
- [x] 输入框自动调整高度
- [x] Enter 发送 / Shift+Enter 换行
- [x] 生成中禁用按钮（防重复提交）
- [x] Typing 加载指示器
- [x] 自动滚动到底部
- [x] 智能滚动 — 仅在用户位于底部时自动滚动，不打断阅读
- [x] 欢迎引导提示
- [x] 错误提示（无 API Key、API 错误、页面提取失败）
- [x] 新建聊天 — 清空会话历史，重新开始对话
- [x] 模型状态栏 — 输入框下方显示当前使用的模型名称
- [x] 复制回复 — AI 消息上一键复制原始文本

- [x] 双语界面 — 中文/英文切换
- [x] 翻译函数 — `t(key, params)` 支持参数插值

- [x] 一键发送 — 点击推荐问题立即发送
- [x] 开关控制 — 可在设置中启用/禁用此功能

- [x] 流式播放 — 使用 MediaSource Extensions 实现流式音频播放
- [x] 句子队列 — 按句子切分，每 2 个句子结尾发送一次

## 已实现（v1.2）

### 夜间模式
- [x] 侧边栏暗色主题 — CSS 变量覆盖，`[data-theme="dark"]` 切换
- [x] 设置页暗色主题 — 硬编码颜色全部迁移为 CSS 变量

- [x] 主题选择器 — 设置页可视化主题卡片选择
- [x] 主题持久化 — `chrome.storage.sync` 存储 `themeName`
- [x] 主题与暗色独立 — 暗色模式与主题配色独立控制

## 已实现（v1.1）

### 选中文本引用
- [x] 实时选区检测 — content.js 监听 selectionchange，300ms 防抖推送
- [x] 引用预览条 — 输入框上方显示选中文本前 50 字预览，可手动清除
- [x] 引用集成 prompt — 自由问答时引用内容作为上下文发给 AI
- [x] 聊天气泡引用预览 — 用户消息中显示引用 blockquote
- [x] 快捷操作适配 — 总结/翻译/关键信息针对选区内容处理

### 快捷指令
- [x] 设置页管理 — 增删改快捷指令（指令名 + prompt 内容），实时保存
- [x] 指令名校验 — 非空、无空格/斜杠、列表内唯一
- [x] 聊天框 `/` 触发 — 输入 `/` 弹出指令列表，实时筛选
- [x] 键盘导航 — `↑`/`↓` 切换选中、`Enter` 执行、`Esc` 关闭
- [x] 指令执行 — 提取页面内容，拼接用户 prompt 发送给 AI

### 消息重试
- [x] 重试按钮 — 最新用户消息上显示刷新按钮
- [x] 重新生成 — 点击后从该消息点重新发送，获取不同回复
- [x] 历史截断 — 重试时清除该消息之后的所有对话
